# ben-api-test
---------------

Project for testing for REST API using Cucumber and Javascript.

Follow below instructions for running the project.


Softwares used:
- Nodejs, 4.x+
- cucumber.js 

Install dependencies:
```bash
$ npm install -g cucumber
```

```bash
$ npm install
```

##  Running the Tests
```bash
$ npm test
```

(or)

```bash
$ npm test features
```
